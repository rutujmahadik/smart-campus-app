# 📱 Smart Campus App - Hackathon Guidelines

Welcome to the Smart Campus App build! [cite_start]We have exactly 4 hours to build this integrated system[cite: 1]. To ensure all 11 modules successfully combine at the end, **every team must strictly follow these rules.**

## 🚨 The Golden Rules

1. **NEVER push directly to the `main` branch.**
2. **ONLY write code inside your team's assigned folder/package.**
3. **DO NOT edit global files** (like `MainActivity.kt` or `themes.xml`) without asking the Project Manager first.

## 🌿 Branching Strategy

When you clone this repository, create a new branch for your team immediately. 

**Branch Naming Convention:** `team-[number]-[module-name]`

*Examples:*
* [cite_start]`team-1-login` [cite: 10]
* [cite_start]`team-3-timetable` [cite: 20]
* [cite_start]`team-8-quiz` [cite: 45]

**Git Commands to start working:**
```bash
git clone [INSERT_YOUR_REPO_URL_HERE]
cd smart-campus-app
git checkout -b team-1-login
